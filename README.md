## EASYC IDE 一个功能简单的c语言编辑器

java编写，gui界面，外观潦草的java课设作品，可供学习参考，下载应付检查之用
- *仅支持windows，使用前请先下载gcc并添加到环境变量*
## 特色功能

- 文件编辑，文件树加载，工作区指定，include路径指定，自动保存，一键编译运行，显示结果，命令行调用

- 相关配置数据均以json文件形式保存

- 受限于技术力限制以及懒狗心态，没有gdb调试功能与代码补全高亮功能
